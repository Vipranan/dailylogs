What kind of 3D scroll experience?

Each has a distinct feel. Pick the one closest to your vision for Maverick.

ALTITUDE: 35,000 FT

A — Full 3D Scene (Three.js / R3F)

A real-time 3D environment: aircraft model flying through clouds/space, camera moves as you scroll. Think Apple Vision Pro intro or Stripe's homepage.

Stack: React Three Fiber + Next.js · Most immersive · Highest effort

SCROLL TO ENGAGE

B — Scroll-Driven 3D Layers (GSAP + CSS)

Elements move in 3D depth as you scroll — parallax layers, rotating cards, 3D text reveals. No heavy engine. Like Linear.app or Framer sites.

Stack: Next.js + GSAP ScrollTrigger · Smooth & fast · Medium effort

DRAG TO ROTATE

C — Embedded Spline 3D (Interactive Object)

A beautiful interactive 3D object (aircraft, globe, AI sphere) embedded in the hero — user can drag to rotate. Rest of site is standard scroll. Like Resend or Luma.

Stack: Next.js + Spline · Easiest 3D · Low-medium effort
